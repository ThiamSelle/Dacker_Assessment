# Partie 6 : Visualisation
## Données sources

Les données sont exportées depuis SQLite vers 4 fichiers CSV dans 'data/dashboard/:
- `kpis_produit.csv` - KPIs détaillés de chaque produit
- `ventes_geographiques.csv` - Distribution complète produit × pays
- `produit_pays_top.csv` - Pays dominant par produit
- `produits_premium.csv` - Top 3 produits par revenu moyen/commande

La Visualisation est disponible au lien suivant: https://lookerstudio.google.com/reporting/135d21aa-48d2-4484-9e68-f92719eb5413.
